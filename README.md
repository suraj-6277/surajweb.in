# 🚀 Suraj Jadhav Portfolio

This is my personal portfolio website built using **Next.js**, showcasing my projects, skills, and journey as a developer.

---

## 🛠️ Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Vercel (Deployment)

---

## ⚡ Getting Started

Run the development server:

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser to see the result.

---

## ✏️ Editing the Project

You can start editing the homepage by modifying:

```
src/app/page.tsx
```

The page auto-updates as you make changes.

---

## 📂 Project Structure

```
src/app        → Pages and routes  
src/components → Reusable UI components  
public         → Static assets   
```

---

## 🌟 Features

* Clean and minimal UI
* Responsive design
* Dark/Light mode support
* Projects showcase
* Blog & personal insights

---

## 📌 Deployment

This project is deployed on **Vercel**.

To deploy your own:

1. Push code to GitHub
2. Import project in Vercel
3. Deploy
